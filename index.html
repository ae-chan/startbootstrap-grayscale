>> Improving the K-Nearest Neighbours (KNN) Algorithm

[Link] A common solution to dealing with missing datapoints is to inject an imputed value via the KNN algorithm. Broadly speaking, it takes the nearest datapoints ("neighbours") and averages their known values to give a single estimate for the datapoint in question. But how exactly are these neighbours identified? The Euclidean formula is the default approach, but that gives equal weighting to each variate. Instead, I adapted the algorithm to use a newly fashioned distance measure. It assigns weights based on each variate's scaled co-efficients from a Multiple Linear Regression cast on the known dataset. Over 10,000 simulations on a complete dataset with missingness being randomized on each run, performance saw an increase but so did run-time. Hence, the new algorithm would best be used in cases demanding of accuracy and where time can be afforded.
Instance-Based Learning, Regression, Classification, Simulations

>> Stock Price Forecasting

[Link] Prophet is Facebook's renowned time-series forecasting algorithm which identifies unique trends segmented by seasonality, holiday effects and other regressors. Yet, I noticed that there were hardly any applications of it to the financial markets; so I developed a tool that integrated Prophet with Yahoo Finance API functionality. With just two user-given parameters, a stock ticker symbol and a forecast horizon, it generates forecasts in a .csv file as well as an interactive html graph for display.
Prophet, Time-Series Forecasting, Yahoo Finance, API

>> COVID-19 in China vs SARS World Total

[Link] Back in February 2020, I wanted to try my hand at the popular data visualization package of ggplot. To challenge myself, I compiled the WHO's reports on the mortality rates of COVID-19 in China and compared them to the aggregated numbers (on similar time scales) as SARS back in '03. The results were alarming, and this was only in February when COVID was yet to be declared a pandemic.
